Counting Barcodes with the same Betti Curve
H. Ashley, H. B. Bjerkevik, J. Curry, R. Decker, R. Green
arXiv preprint · 10.48550/arXiv.2602.09011
arXiv →
Abstract

This paper considers an important inverse problem in TDA: how many different barcodes produce the same Betti curve? Our answer connects persistent homology to Kostant's partition function and, surprisingly, to enumerative combinatorics for so-called "magic" juggling sequences studied by Ronald Graham and others—proving an equivalence between the inverse problem and corresponding statements in these two other settings. From an applications perspective, the work provides a quantification of how lossy the TDA pipeline is when moving from persistent homology to persistent Betti numbers.

Computational Complexity of the Interleaving Distance
Applied Topologists in Albany (ATiA) Seminar · University at Albany, SUNY · March 6, 2026
Abstract

This talk discusses the computational complexity of several data structures used in topological data analysis, particularly merge trees and Reeb graphs. Proofs by Bjerkevik, Botnan, Dey, Erickson, and others establish that computing the interleaving distance between merge trees and between Reeb graphs is NP-complete, computing isomorphism between Reeb graphs is GI-complete, and computing isomorphism between merge trees can be done in linear time.
